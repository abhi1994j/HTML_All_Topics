<dialog> Tag

<dialog> Element

This is a native HTML5 <dialog> element — no extra libraries needed!


It comes with a built-in ::backdrop pseudo-element for the overlay, and the open attribute controls visibility.

<dialog> creates a native modal or popup window.
.showModal() → opens as a modal with a backdrop overlay.
.show() → opens without backdrop (non-modal mode).
.close() → closes the dialog.
::backdrop → CSS pseudo-element for styling the overlay.

draggable="true" → makes any element draggable by the user.
draggable="false" → explicitly disables dragging (default for most elements).
Works with JS drag events: dragstart, dragover, drop, dragend.
Images and links are draggable by default — no attribute needed.

contenteditable="true" → makes any element's content directly editable in the browser.
contenteditable="false" → explicitly prevents editing (default behaviour).
spellcheck="true" → enables browser spell-check on editable content.
document.execCommand() → applies formatting (bold, italic, underline) to selected text.

<meter> represents a scalar value within a known range (unlike <progress>).
min / max → the full range of the meter.
low → threshold below which value is considered "low".
high → threshold above which value is "high".
optimum → the ideal/best value — browser colors green/yellow/red automatically.

<canvas> Tag

<canvas> is a bitmap drawing surface controlled entirely via JavaScript.
getContext('2d') → returns the 2D drawing API.
beginPath(), moveTo(), lineTo(), stroke() → draw lines.
Use cases: games, charts, image editing, animations, signatures.

<template> Tag

<template> holds HTML that is not rendered until cloned via JavaScript.
document.querySelector('template').content → accesses the template's DOM.
cloneNode(true) → makes a deep copy ready to insert into the page.
Perfect for repeating UI patterns like cards, list items, or table rows.
